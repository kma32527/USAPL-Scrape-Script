# USAPL-Scrape-Script
A script to scrape data from USA Powerlifting databases into a .csv file
Variable key:
 ['name', 'yob', 'club', 'state', 'weight', 'squat1', 'squat2', 'squat3', 'bench1', 'bench2', 'bench3', 'dead1', 'dead2', 'dead3',
             'total', 'points', 'date', 'meetid', 'location', 'director']
